# HyperCRX-InsightPilot

HyperCRX-InsightPilot 是一款面向 GitHub / Gitee 的智能浏览器扩展，通过引入大语言模型（LLM）能力，实现从传统“静态可视化”向“可对话、可解释、可行动”的开源运营助手转变。
项目已在 GitHub 开源，旨在帮助开发者、维护者和开源社区管理者高效理解项目动态、获取洞察并支持数据驱动决策。

### **1. 多平台数据可视化（GitHub / Gitee）**

- 支持对仓库、开发者信息的自动识别与分析

- 无缝嵌入平台界面，无需跳转即可查看数据洞察

- 提供活跃度、OpenRank、贡献者行为等多维指标

- 包含工具提示、趋势图、动态图表等扩展功能

  （Issue Tooltip、PR Tooltip、活跃度趋势、开发者行为分析等）

### **2. LLM 驱动的语义分析与趋势解读**

通过结合指标语义模板与大语言模型，系统可自动：

- 对关键指标进行趋势分析
- 识别异常波动
- 输出自然语言说明

示例：

> 过去 6 个月项目活跃度持续上升，社区提交与讨论频率显著提高。

### **3. 自动化报告生成（Insight → Report）**

- 一键生成完整项目分析报告：指标总览、趋势图表、AI 解读、运营建议
- 自动导出 Markdown，可直接用于 README、会议文档或团队平台
- 内置 **HyperEval** 自评估框架：检测幻觉、逻辑冲突、数据异常，提高报告可信度

###  **4. 自然语言问答（ChatOps for Open Source）**

系统支持针对仓库进行自然语言提问，例如：

- 为什么最近活跃度下降？
- 帮我分析 Star 增长的原因。
- Issue 响应效率是否出现变化？

系统会结合多维数据进行推理，并提供可解释的洞察建议，实现从数据到决策的完整闭环。
